signalguard-logs

Log intelligence and AIOps: parsing, template extraction, bursts, new patterns, semantic anomalies.

Python package Sister toolkit to signalguard-aiops

What it is

signalguard-logs is a log AIOps toolkit for structured parsing, template extraction, TF-IDF encoding, and anomaly detection on logs: volume bursts, new error patterns, and semantic outliers.

Core use cases

• Detect bursts of ERROR logs for a service
• Flag previously unseen error templates
• Identify semantically unusual log messages
• Combine volume, templates, and semantics into one "log health" signal

Tech stack

Python · NumPy · Pandas · scikit-learn · Regex / JSON parsing · TF-IDF · IsolationForest.

Quick start

Install locally, parse your logs into LogStream, and plug them into high-level recipes.

Install CLI 
git clone https://github.com/danial-amin/signalguard-logs
cd signalguard-logs
pip install -e .
Parse logs Parsing 
from signalguard_logs.models import LogStream
from signalguard_logs.parsing import RegexLogParser

parser = RegexLogParser()
with open("app.log") as f:
    records = list(parser.parse_lines(f))

stream = LogStream(records)
Error burst recipe Incidents 
from signalguard_logs.recipes import ErrorBurstRecipe

recipe = ErrorBurstRecipe(service="checkout", level="ERROR")
result = recipe.run(stream)
burst_labels = result["labels"]
New pattern recipe Patterns 
from signalguard_logs.recipes import NewErrorPatternRecipe

recipe = NewErrorPatternRecipe(service="auth", level="ERROR")
result = recipe.run(stream)
new_pattern_labels = result["labels"]

Detectors

LogBurstDetector · NewTemplateDetector · SemanticIForestDetector

LogBurstDetector groups logs into time windows and flags unusually high counts of ERROR-level records.

NewTemplateDetector uses simple template extraction to flag previously unseen error patterns, ideal for catching new failure modes.

SemanticIForestDetector encodes messages with TF-IDF and uses IsolationForest to surface outlier messages.

Combined log health

Volume × Template novelty × Semantic outliers.

With CombinedLogHealthRecipe, you can produce a single per-log anomaly score that blends bursts, new patterns, and semantic outliers — ready to be surfaced next to metric-based incidents from signalguard-aiops.
